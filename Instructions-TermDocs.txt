Open Terminal:
	Type: sudo cp "path to termdocs" /usr/bin/
	hit enter: Enter Your Password
	hit enter:
	Type: sudo chmod +x /usr/bin/TerminalDocs
Open Terminal:
	Type: termdocs "Path" 
    -To open Specified Path or create new File
	hit enter: The program should now be running.
	hit enter: Choose this file
	Type: -h for help menu
